<task type="auto">
  <name>Task 1: Configure Tailwind theme with western color palette and typography</name>
  <files>tailwind.config.ts, src/app/globals.css</files>
  <action>
  Update tailwind.config.ts to extend the default theme with:

  **Colors** (under `theme.extend.colors`):
  - `navy: { 900: '#0A1219', 800: '#0F1923', 700: '#162A3E', 600: '#1E3A5F' }` — background hierarchy
  - `gold: { 500: '#C8A55C', 400: '#D4B76A', 300: '#E0C97E' }` — western accent
  - `teal: { 500: '#2A9D8F', 400: '#35B8A8' }` — secondary accent
  - Keep default slate colors for text (slate-300, slate-400, slate-500 map well to text hierarchy)

  **Typography** (under `theme.extend.fontFamily`):
  - `heading: ['Montserrat', 'system-ui', 'sans-serif']` — bold western headings
  - `body: ['Inter', 'system-ui', 'sans-serif']` — clean readable body text

  **Font sizes** (under `theme.extend.fontSize`) — custom scale for western authority:
  - `'display': ['3.5rem', { lineHeight: '1.1', fontWeight: '800' }]` — hero headlines
  - `'heading-1': ['2.25rem', { lineHeight: '1.2', fontWeight: '700' }]`
  - `'heading-2': ['1.75rem', { lineHeight: '1.25', fontWeight: '700' }]`
  - `'heading-3': ['1.25rem', { lineHeight: '1.3', fontWeight: '600' }]`

  **Spacing/radius:**
  - No custom spacing needed — Tailwind defaults work well
  - Add `borderRadius: { 'card': '0.5rem' }` for consistent card rounding

  In globals.css, add:
  - CSS custom properties for the color values (for use outside Tailwind contexts)
  - Set `body` background to navy-800, text to white, font-family to body font
  - Import Google Fonts: Montserrat (600, 700, 800) and Inter (400, 500, 600) via `@import` at top of file
  - Set `::selection` colors to gold on navy for branded text selection
  - Smooth scrolling on `html`
  </action>
  <verify>npm run build succeeds; inspect tailwind.config.ts to confirm all custom tokens are present; check globals.css for font imports and CSS custom properties</verify>
  <done>Tailwind config has all custom colors (navy, gold, teal), font families (heading, body), custom font sizes (display, heading-1/2/3), and card border radius. globals.css imports Montserrat and Inter, sets dark background and white text default, has CSS custom properties defined.</done>
</task>
